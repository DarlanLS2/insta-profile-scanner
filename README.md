## Requirements
 - Node.js >= 20
 - npm
 - Chromium
 - Linux

## Installation: 
``` bash
git clone https://github.com/DarlanLS2/insta-profile-scanner
cd insta-profile-scanner
npm i
```

## Usage: 
``` bash
chmod u+x ./main.sh
./main.sh
```

### How to start chromium: 
``` bash
chromium --remote-debugging-port=9222
```
